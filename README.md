# Rule-based система выбора терапии АГ

Это статический SPA-проект без сборщика: достаточно распаковать архив и открыть `index.html` в браузере.

## Что внутри
- 2 страницы: форма ввода и страница рекомендаций.
- Rule-based движок с конфигурацией сценариев и шагов терапии.
- Отдельные модули:
  - `src/config` — правила и каталог препаратов
  - `src/logic/calculators.js` — расчёты степени, стадии, риска, баллов, целевых уровней АД
  - `src/logic/validation.js` — проверки ошибок и противоречий
  - `src/logic/therapyEngine.js` — выбор сценария и терапии
  - `src/logic/explanation.js` — блок «Почему рекомендовано именно это»
  - `src/logic/conclusion.js` — генерация краткого заключения
- Скрытый редактор правил: нажмите `Ctrl + Shift + R` на первой странице.

## Ограничения
- Для SCORE2 добавлены локальные заглушки-иллюстрации, так как исходные изображения не были приложены.
- В таблице CHA2DS2-VASc для 5 и 8 баллов в исходном описании значения частоты инсульта были указаны неоднозначно (`6,?`). В интерфейсе это явно помечено как «значение требует уточнения в конфиге».

## Как использовать
1. Откройте `index.html`.
2. Заполните форму.
3. Нажмите `Назначить терапию`.
4. На второй странице посмотрите рекомендации, объяснение логики и сгенерируйте краткое заключение.

## Для GitHub
Можно просто загрузить всё содержимое архива в новый репозиторий и включить GitHub Pages.
